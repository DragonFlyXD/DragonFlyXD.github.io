【Nginx】Location匹配详解

发表于 2017-03-28 | 分类于 Nginx
字数统计 332 | 阅读时长 1

定义

location 是定义资源类型与服务器中资源地址 uri 的映射关系。

语法

location [=|~|~\*|^~|@] /uri/ { … }

= 严格精确。不会进行匹配正则。

~ 区分大小写。

!~ 区分大小写不匹配。

~* 不区分大小写。

!~* 不区分大小写不匹配。

^~ 不继续匹配正则。

@ Named Location 用来处理内部重定向。
