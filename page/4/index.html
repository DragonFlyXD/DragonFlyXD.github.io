【Mysql】使用Navicat Premium连接MAMP中的mysql

发表于 2017-03-28 | 分类于 Mysql
字数统计 138 | 阅读时长 1

由于Navicat Premium默认连接是本地的mysql数据库，因此要用它连接MAMP中的mysql需要作些配置。

解决步骤如下：

  • 新建一个mysql连接，将常规中的内容填好，主机名或IP地址务必选择localhost。

  • 选择高级,将使用套接字文件的复选框勾选上，填上的内容如下：

    /Applications/MAMP/tmp/mysql/mysql.sock

注意，如果你填写的主机名是127.0.0.1,连接时会报出hostnamemustbelocalhostwithsocketfile的错误。

即套接字文件支持的主机名必须为localhost。

Introduction

  • PHP数据对象(PHP Data Objects)扩展为PHP访问数据库定义了一个轻量级的一致接口。
  • 实现了PDO接口的每个数据库驱动可以公开具体数据库特性作为标准扩展功能，也就是说实现了PDO接口后不仅能访问mysql一种数据库,其他数据库亦能。
  • PDO提供了一个数据访问抽象层，这意味着，不管使用那种数据库，都可以用相同的函数(方法)来查询和获取数据。另外，PDO不提供数据库抽象层。

三个类

  • PDO类，与数据库连接有关(连接、执行sql语句)。
  • PDOStatement 处理结果集。
  • PDOException 异常处理类。

符号->、=>和::三者的区别

定义：

  • ->,插入式解引用操作符(infix dereference operator),将对象的引用指向对象，例如对象->对象的方法。
  • =>,一般用于数组，键名=>键值。
  • ::,作用域解析操作符(Scope Resolution operator),用于调用类的静态成员变量或是类之间的调用。

this、self、parent三者的区别

定义：

  • this是指向当前对象的指针。
  • self是指向当前类的指针。
  • parent是指向父类的指针。

【PHP】会话控制

发表于 2017-03-28 | 分类于 PHP
字数统计 1,769 | 阅读时长 6

Background

  • HTTP是无状态的协议，这是说，HTTP协议没有一个内建机制来维护两个事物之间的状态。当一个用户在请求一个页面后再去请求另一个页面，HTTP将无法告诉我们这两个请求是来自同一个用户。
  • 会话控制的思想是指能够在网站中根据一个会话跟踪用户。如果我们做到了这一点，对于用户登录的支持就很容易做到，并根据其授权级别和个人喜好来显示相应的内容。
  • PHP的会话是通过唯一的会话ID来驱动的，会话ID是一个加密的随机数字。会话ID由PHP生成，在会话的生命周期中保存在客户端。它可以保存在用户机器里的cookie中，或者通过URL在网络上传递。
  • 会话ID就像一把钥匙，它允许我们注册一些特定的变量，称为会话变量。这些变量的内容保存在服务器端，会话ID是客户端唯一可见的信息。
