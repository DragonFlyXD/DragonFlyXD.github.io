【Laravel】数据库迁移时出现No such file or directory情况

发表于 2017-03-28 | 分类于 Laravel
字数统计 162 | 阅读时长 1

php artisan migrate

最有可能原因：(一般在MAMP套件中出现)php命令使用的mysql socket是本地的mysql中的,没有在配置文件中配置MAMP的mysql socket。

在config/database.php中mysql配置项里添上unix_socket即可。

其他原因：

  • config/database.php与./.env文件中的mysql配置不正确。

  • 尝试将DB_HOST由localhost改成127.0.0.1。

    'host' => env('DB_HOST', '127.0.0.1') (config/database.php)

    DB_HOST=127.0.0.1 (.env)

  • 将php.ini中的mysql.default_socket指向正确的地址。

    mysql.default_socket = /Applications/MAMP/tmp/mysql/mysql.sock。
