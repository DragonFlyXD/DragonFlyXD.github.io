MongoDB是时下比较流行的NoSQL数据库。

与传统数据库(如mysql,oracle等)相比，它具有高扩展性与高性能(能储存于物理内存中)，最重要的是Json的存储类型。

以下简述在Mac中如何使用MongoDB

  • 安装MongoDB。

    推荐使用Mac下的包管理工具homebrew安装。

    在终端中执行 brew install mongodb 命令。
