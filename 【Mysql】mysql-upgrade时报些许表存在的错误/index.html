【Mysql】mysql_upgrade时报些许表存在的错误

发表于 2017-03-28 | 分类于 Mysql
字数统计 402 | 阅读时长 2

今天在给MAMP的mysql使用 mysql_upgrade 升级时,出现了以下错误：

(原因不确定)经过网上一番查找资料:数据库打开这些表的默认引擎是MyISAM，但这几张表的创建时的默认引擎是InnoDB。

以下为解决步骤：

  • 登录mysql。

  • 进入mysql数据库，删除这些表。

  • 停止mysql服务，并进入到数据库文件所在目录，删除与之相关的ibd文件。

  • 重启服务，进入mysql数据库，重建以上被删除的表。
