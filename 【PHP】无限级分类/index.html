【PHP】无限级分类

发表于 2017-03-28 | 分类于 PHP
字数统计 838 | 阅读时长 3

具体应用

  • 查找指定栏目的子孙栏目，即子孙树。
  • 查找指定栏目的父栏目/父父栏目……顶级栏目，即家谱树。

案例素材

  • $area数组记录了地区的等级分类。
  • 数组的parent的值指向了父地区的id值，例如，温州的parent的值为1，则id值为1的浙江就是其父地区。

先来个easy的，找子栏目

行，来个进阶的，找子孙树

Summary

  • 函数就是个封装的执行体，前后执行，并无联系。
  • 在函数中声明的static静态变量，无论此函数被调用多少次，只初始化一次。以后直接沿用该变量，在递归时，十分受用。
  • 如果不想使用static，可以使用array_merge函数,例如：seekChild($arr, $value['id'])改为$children=array_merge($children,seekChild($arr,$value['id']))。
  • 注意，在这里，数组合并时使用+号是不行的，因为+时，两个数组键名相同并不会合并。

static总结

  • 修饰类的属性和方法为静态属性，静态方法。
  • static::method()，延迟绑定。
  • 在函数/方法中，作为声明静态变量使用。
