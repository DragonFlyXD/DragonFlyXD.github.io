后经一番排查 ，原来是我将 PATH 指定成唯一的，从而检索不到其他目录的命令。

如 export PATH=/Users/chenlongfei/Documents/mygo/bin 这样设置后。

在Zsh输入命令后只会在 /Users/chenlongfei/Documents/mygo/bin 路径下找，因此会提示找不到命令。

正确的设置方法应该是在路径前或路径后添加 $PATH 变量。
